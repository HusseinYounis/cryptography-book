# Introduction to Cryptography - Interactive Textbook

[![Deploy Book](https://github.com/yourusername/cryptography-book/actions/workflows/call-deploy-book.yml/badge.svg)](https://github.com/yourusername/cryptography-book/actions/workflows/call-deploy-book.yml)

An interactive textbook for learning cryptography, built with [TeachBooks](https://teachbooks.io/).

## üîí About

This interactive book provides a comprehensive introduction to cryptography, covering:

- Classical cryptography and cryptanalysis
- Mathematical foundations (number theory, modular arithmetic)
- Modern symmetric encryption (AES, block ciphers, modes of operation)
- Public key cryptography (RSA, Diffie-Hellman, Elliptic Curves)
- Hash functions and digital signatures
- Practical implementations with Python
