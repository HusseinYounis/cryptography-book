# Logo and Favicon Information

## Logo Files Needed

Please add the following logo files to the `book/figures/` directory:

1. **logo_light.png** - Logo for light mode (recommended size: 200x200px)
2. **logo_dark.png** - Logo for dark mode (recommended size: 200x200px)
3. **favicon.ico** - Browser tab icon (16x16px or 32x32px)

## Temporary Solution

Until custom logos are created, you can:
1. Use free icons from [Font Awesome](https://fontawesome.com/)
2. Create simple logos using [Canva](https://www.canva.com/)
3. Use open-source icons from [The Noun Project](https://thenounproject.com/)

## File Formats

- **Logo files**: PNG with transparent background
- **Favicon**: ICO format or PNG (will be converted)
